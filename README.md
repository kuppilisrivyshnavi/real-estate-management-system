# 📌 Project Description

The **Real Estate Management System** is a full stack web application developed to manage property listings efficiently.  
This system allows administrators to add, edit, view, and delete real estate properties through a clean dashboard interface.

The application is built using **Python Flask for backend development**, **SQLite for database management**, and **HTML, CSS, and Bootstrap for the frontend user interface**.

This project demonstrates a basic full stack architecture with database integration and dynamic web page rendering.

# ✨ Features

- Admin Login System

- Add New Properties

# ⚙️ Installation

### 1️⃣ Clone the repository


git clone https://github.com/kuppilisrivyshnavi/real-estate-management-system.git


### 2️⃣ Navigate to project folder


cd real-estate-management-system


### 3️⃣ Install dependencies


pip install -r requirements.txt


### 4️⃣ Run the application


python app.py


### 5️⃣ Open in browser


http://127.0.0.1:5000
